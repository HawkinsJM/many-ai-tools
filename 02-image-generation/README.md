# 02 — Image Generation

Generates an image with Gemini's Nano Banana model (`gemini-2.5-flash-image`) when a button is clicked, and displays it on a p5 canvas. Also includes the chat-with-a-cat feature from demo 01.

## Setup

1. `npm install`
2. Create a `.env` file with:
   ```
   GEMINI_API_KEY=your_key_here
   ```
3. `npm run dev`
4. Open http://localhost:3000

## Get API keys

- Gemini: https://aistudio.google.com/apikey
